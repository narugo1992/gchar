Installation
===================

``gchar`` is currently hosted on PyPI. It required python >= 3.7.

You can simply install gchar from PyPI with the following command:

.. code:: shell

    pip install gchar

You can also install with the newest version through GitHub:

.. code:: shell

    pip install -U git+https://github.com/narugo1992/gchar.git@main

After installation, run this python code, and version information \
of ``gchar`` should be shown.
