Supported Sites¶

We currently support searching on pixiv, where we can generate the best search keywords based on the known character’s name and an online pre-processing database (see gchar.resources.pixiv.get_pixiv_keywords()).

In addition to that, we also support 14 popular image websites and can retrieve recommended search terms for known characters on these websites using the gchar.resources.sites.get_site_tag() function. The following are the currently supported websites:
